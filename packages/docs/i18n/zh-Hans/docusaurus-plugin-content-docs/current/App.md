## 如何运行

1. 安装 expo `yarn global add expo-cli`
2. 安装依赖 `yarn install`
3. 启动编译 `expo start`
4. 根据控制台输出的提示, 在模拟器或者真实设备上运行 app

想要了解更多信息, 请查看 [https://docs.expo.io/](https://docs.expo.io/)
